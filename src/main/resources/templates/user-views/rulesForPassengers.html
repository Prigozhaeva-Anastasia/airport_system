Правила поведения в самолете. Инструкции по безопасности пассажиров
Согласно статистическим данным, каждый второй турист для путешествий выбирает воздушные судна. Такой способ передвижения позволяет экономить время. Также он считается самым безопасным. Авиакомпании установили свои правила поведения в самолете. Их должен соблюдать каждый, чтобы не возникало неприятных ситуаций.
